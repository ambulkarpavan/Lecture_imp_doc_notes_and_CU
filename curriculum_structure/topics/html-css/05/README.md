# Extensions (Basic)  (2-3mins)

## The Importance of Making File Extensions Visible in Web Development

When you're diving into web development, there are many small details that can make a big difference in your workflow, one of which is making file extensions visible. By default, many operating systems hide these extensions in an effort to simplify the user interface. However, for a web developer, this can be more confusing than helpful. In this article, we'll explore the importance of making file extensions visible and guide you through enabling this feature on Windows, macOS, and Linux.

### Why File Extensions Should Be Visible

1. **Avoid Mistakes**: It's easy to accidentally save an HTML file as `.txt` or a JavaScript file as `.java` when you can't see the extension.
2. **Quick Identification**: Developers often work with multiple file types—HTML, CSS, JavaScript, and more. Quick identification of each file type is essential for a smoother workflow.
3. **Security**: Some malicious files masquerade as benign file types. Being able to see the full file extension can offer an additional layer of security.
4. **Better Organization**: When working in a directory full of files, being able to sort and manage them by extension can be incredibly helpful.

### How to Make File Extensions Visible

### Windows

1. Open any folder using File Explorer.
2. Click on the "View" tab. If you're on Windows 10, you may need to click the "Show/Hide" panel to see the options.
3. Check the "File name extensions" box.

In Windows 11:

1. Open File Explorer.
2. Click on the three-dot menu (ellipsis) in the toolbar.
3. Choose "Show" and toggle on "File name extensions."

### macOS

1. Open Finder.
2. Go to Finder > Preferences from the menu bar.
3. Click the "Advanced" tab.
4. Check the box that says "Show all filename extensions."

### Linux (Ubuntu with Nautilus)

1. Open the Nautilus file manager.
2. Go to the menu in the top-right corner (three horizontal lines) and select "Preferences."
3. In the "Views" tab, check the box that says "Show hidden and backup files."
4. Additionally, you can use the `ls -a` command in the terminal to list all files, including those with extensions, in a directory.
