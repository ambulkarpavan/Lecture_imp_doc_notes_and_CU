- For each session, assignments can be accessed in the following directory: `programs/<program-code>/<course-code>/sessions/<session-number>/class/assignments/README.md`. The README.md file contains a Master Template Link, which compiles all the assignments designated for a specific session.

- Additionally, detailed information for each assignment problem, including the problem statement and its solution, is provided in separate files located at `programs/<program-code>/<course-code>/sessions/<session-number>/class/assignments/problem-1.md`. Each file is organized with two main sections:

  - Problem Statement
  - Solution

- Assignment problems are individually named following a specific naming convention for easy reference:

  - problem-1.md ( Time taken to solve each problem should also be a part of this README )
  - problem-2.md ( Time taken to solve each problem should also be a part of this README )
  - problem-3.md ( Time taken to solve each problem should also be a part of this README )
