## Quiz 3

What is the role of the .catch() method in a fetch request?

Options

A. To define a new function.

B. To create a loop.

C. To handle errors gracefully, in case something goes wrong during the request.

D. To convert the response to JSON format.

**Solution (Correct Option + Explanation)**

Ans. C => To handle errors gracefully, in case something goes wrong during the request. The .catch() method in a fetch request is used to handle errors gracefully, in case something goes wrong during the request.
