## Quiz 1

In JavaScript, what is the purpose of the fetch() function?

Options

A. To define a new function.

B. To create a loop.

C. To make network requests and handle responses.

D. To manipulate strings.

**Solution (Correct Option + Explanation)**

Ans. C => To make network requests and handle responses. The fetch() function in JavaScript is used for making network requests, such as fetching data from a server or sending data to one.
